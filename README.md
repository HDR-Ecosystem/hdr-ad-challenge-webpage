# HDR Anomaly Detection ML Challenge Website [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19392603.svg)](https://doi.org/10.5281/zenodo.19392602)


This repository generated the website for the first HDR ML Challenge, _Anomaly Detection in Scientific Domains_, and its associated workshop and award ceremony at AAAI25.

At the time, this repository was embedded as an iframe in the HDR website; however, since the completion of the challenge, this website has been integrated directly into the [HDR website source code](https://github.com/HDR-Ecosystem/hdr-website), under the [mlchallenge-y1 directory](https://github.com/HDR-Ecosystem/hdr-website/tree/main/html/mlchallenge-y1).
